📱 Debt & Bill Manager for iOS

A Swift‑powered iOS app designed to help users organize bills, track payments, and manage debt installments with clarity and control.

🚀 Overview

This app provides a clean and intuitive interface for managing recurring expenses and structuring debt payments. Users can register bills, track due dates, split debts into installments, and visualize their financial commitments in a simple, actionable way.

✨ Features

Bill Management – Add, edit, and track recurring or one‑time expenses.
Debt Installment Planning – Split debts into custom installments and monitor progress.
Payment Tracking – Mark payments as completed and keep your financial history organized.
Swift‑Native Architecture – Built entirely in Swift for performance and seamless iOS integration.
Modern UI – Clean, minimal interface focused on usability and clarity.

🛠️ Tech Stack

Language: Swift
Framework: SwiftUI
Platform: iOS
Architecture: MVVM

📦 Installation

Clone the repository and open the project in Xcode:

bash
git clone https://github.com/JBOliveira-pt/GestorContas.git
cd GestoContas
open *.xcodeproj
Run on an iOS Simulator or physical device.
