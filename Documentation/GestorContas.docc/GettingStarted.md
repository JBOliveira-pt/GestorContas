# Getting Started

## Requisitos

- Xcode (macOS) para compilar e executar
- iOS 16+ (ajuste conforme o deployment target do projeto)

## Como executar

1. Abra `GestorContas.xcodeproj` no Xcode
2. Selecione um simulador (ex.: iPhone 15)
3. Pressione **Run**

## Estrutura do projeto

- `Views/` — telas SwiftUI
- `ViewModels/` — lógica de apresentação (MVVM)
- `Models/` — modelos de dados
- `Services/` — serviços e regras auxiliares
